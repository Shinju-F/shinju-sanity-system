# Changelog

Все значимые изменения в системе Shinju Sanity System будут документированы в этом файле.

## [1.0.0] - 2024-01-01

### Добавлено
- Первоначальный релиз системы рассудка
- Полная интеграция с D&D 5e
- Система из 5 уровней стресса
- База данных из 30+ состояний (психозы и воодушевления)
- Панель управления для мастера
- Персональная панель для игроков
- Библиотека состояний
- Автоматические сообщения в чат
- Настройки модуля
- Темное фэнтези оформление
- Адаптивный дизайн

### Технические особенности
- Application V1 API
- Хранение данных через флаги акторов
- Модульная архитектура
- Handlebars шаблоны
- Кастомные CSS стили
